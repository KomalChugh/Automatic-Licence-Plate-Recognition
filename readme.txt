Running Instructions:-

Have matlab and its image processing toolbox installed.

For single image : 
Open the file run.m in the 'single' folder. Specify the jpg image path (without extension) in the first line of matlab code, 'image='.
Then run the file. The image and its corresponding extracted plate will be displayed and the output result will be stored in 'result.txt' file.
